Waves


This project creates a bunch of randomly-generated sine waves made of colorful circles

Created on September 24, 2020

Source Code [Processing]

About the code


Problem:

Create a visual representation of a sine function


Approach:

I created a file that uses a for loop to create a circle at every value of sin() that exists on the screen. I also used perlin noise to change the rate at which the size of the circles change as they go through the sine function. Compared to the random function, perlin noise makes the sin waves seem smooth and natural instead of chaotic and random.


Challenges:

A challenge that I encountered while creating this project is making the sine waves repeat themselves after they have passed the screen. To overcome this challenge, I created a for loop that randomized the location at which the sine wave begins and also adds a random phase angle to the sine wave so that all of the waves will not be parallel to each other.


Reflection:

This project helped me understand the benefits of perlin noise, and its differences from the random() function.
