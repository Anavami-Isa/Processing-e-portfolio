Parallax Perspective


This project creates a scence on the prairie, that changes perspective as you move you cursor.... it's almost as if you're actually there!

Created on September 10, 2020

Source Code [Processing]

About the code


Problem:

Create a display that utilizes the parallax effect to create depth


Approach:

To do this, I first drew the foreground, middleground, and background. I then made a variable, offset, that holds the value of the distance of the cursor from the center of the screen. The foreground was then moved according to this value. The middleground and background were being moved by fractions of that variable. The farther back the object, the smaller the fraction.


Challenges:

A challenge that I encountered was making the background stand out from the foreground. To do this, I set the offset of the mouse from the center of the screen to a variable. I then shifted everything in the foreground according to that variable. To make the background stand out, I made the background move at only a fraction of that variable, making the background move slower, giving the illusion of depth.


Reflection:

This project helped me understand the importance of speed when creating visual displays such as this. By simply changing the speed at which an object moves, I was able to create the illusion of depth.
