Highway


This project creates a scene of a highway with buildings on the side, moving towards ou as you press the 'up' arrow. It's just like a relaxing drive down the freeway :).

Created on September 29, 2020

Source Code [Processing]

About the code


Problem:

Create a display that simulates a car driving down a road where you are in the perspective of the driver


Approach:

I created a class that held an array of buildings, each with a random height. I then built a method that replaced every building that moves out of view.


Challenges:

A challenge that I came across was making all of the buildings move down the road and replace themselves at the top of the landscape. To fix this, I multiplied the x and y positions of all buildings by a constant that simualted their linear motion.


Reflection:

This project helped me better understand how to use arrays in classes. This project also helped me understand how to move multiple objects at the same time using for loops and methods for movement.
