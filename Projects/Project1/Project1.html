Conic Sections


This project makes conic sections with a circle

Created on October 22, 2020

Source Code [Processing]

About the code


Problem:

Create conic sections around a circle


Approach:

I used the perpendicular line formula to find the equation of each line.


Challenges:

A challenge that I encountered was having the lines drawn for only one side of the circle. To overcome this, I had to create more code that copied the lines to apply to the other side of the circle. That other side of the circle makes the lines appear like an hourglass instead of the parabolic shape that will form if a line was used instead of a circle.


Reflection:

This project helped me better understand how to construct lines and shapes and manipulate their positions.
