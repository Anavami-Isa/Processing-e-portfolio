Array Boxes


This project creates an array of boxes that change color when you hover over them

Created on September 10, 2020

Source Code [Processing]

About the code


Problem:

Create a display that symbolizes an array where each element of the array reacts to the cursor floating on it.


Approach:

To complete this project, I divided the given area into a certain number,x, of sections. I then divided the width and height of the area by x to get the area of each individual square. After that, I simply created a yellow circle that follows the cursor and an if statement that turns the squares black when the cursor hovers over them.


Challenges:

A challenge that I encountered while completing this project was being able to define the area of each square, especially those that are not on the edge of the display. To overcome this challenge, I simply created a for loop with an increment variable,i, that is multiplied by the width or height in order to know where on the board the square is. For example, If I wanted to know the y value of the 3rd square from the top, I would just simply multiply the height of each square by i, which would be 3.


Reflection:

This project helped me understand how to use for loops and if statements to create visual displays such as this.
