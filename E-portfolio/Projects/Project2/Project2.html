Rotating Fractal


This project creates a fractal of circles.... and it rotates!

Created on October 1, 2020

Source Code [Processing]

About the code


Problem:

Create a fractal that rotates using pushMatrix to translate and rotate shapes


Summary

This project 1) uses recursion to create the many layers of the fractal and 2) translate() and rotate() to make the whole fractal rotate about the center of the display


Approach:

I used both recursion and pushMatrix to complete this project


Challenges:

A challenge that I encountered while making this is understanding the pushMatrix() and translate functions. Through this project I learned how pushMatrix isolates functions meant for certain shapes and that the translate function does not move shapes, but rather moves the origin at which the shapes are drawn.


Reflection:

By completing this project, I learned how to use recursion and I also learned how to efficiently use pushMatrix and shape manipulation functions such as rotate and translate.
