Drawing with Lines


This project takes an image and draws it.... using lines!

Created on December 14, 2020

Source Code [Processing]

About the code


Problem:

Create a project that creates an image


Approach:

To try to cover the face of the display, I decided to use lines that span the screen. To create these lines, I made a method that chooses two random points on the display and connected them. Next I used the slope formula to find the slope of the line that was just created. I then use this slope to draw small circles across that line. The position of each of the circles is stored in a variable and the color at that positon is recorded. Each circle is then filled with the color value of the image at its position. This line of circles will then subsequently reveal a portion of the image and eventually, the whole image would be revealed. I also made the image black and white to emphasize the colors being created by the circles.


Challenges:

A challenge that I encountered was determining the length of each line. The lines always came out to be too short, or too long. I overcame this challenge by dividing the distance covered by the line(found using the distance formula) by the slope of the line.


Reflection:

This project helped me better underestand methods in Processing that had to deal with color and image manipulation.
